Heavily inspired by Skeleton, this CSS boilerplate contains a simple CSS reset, base html styles using modern techniques, and a few features to help things along. It enables smooth scrolling, enforces the hidden attribute, and adds a visually-hidden class to keep hidden content available for screen readers.

This is not a framework, but a starting point. There is no build, no package, no hastle. Just download the file and start adjusting it to fit your needs. What is here has been designed to be as little opinionated style as possible while giving a nice baseline aesthetic.

Typography

The typographical base is built on a 16px base (1rem) on a 1.25 modular scale. All font sizes are set with rems and have custom line heights.

The fonts are Open Sans, with Raleway for headings, and Fira Code for monospace. Fonts can be changed using CSS custom properties.

Other semantic type tags like anchors, strong, emphasis, underline, small, del, ins, mark, abbr with a title, q with cite are all included.

Blockquote

A blockquote can be created using the <blockquote> element, or by using the .callout class. Paragraph elements should be placed inside the blockquote element.

Code

Styles have been included for <code>, <samp>, and <kbd> elements. Code blocks can be made by wrapping a <code> element inside a <pre> element. Code blocks overflow automatically on the x axis in case there is not enough room for a line of code.

.some-class {
  background-color: blue;
}

<!-- Remember every whitespace and break will be preserved in a <pre>, including indentation in your code. -->

CSS Variables
Variable Description Default
--font-body Sets the default font Open Sans
--font-head Sets the font for headings Raleway
--font-mono Sets the font for code Fira Code
--color-primary Sets the primary color #4299E1
--color-accent Sets the secondary color var(--color-primary)
--element-bg-color Color used for element backgrounds #EDF2F7
--text-color Color used for all text #1A202C
--heading-color Color used for headings var(--text-color)
--light-text-color Color used for rules and tables #A0AEC0
--link-color Color used for navigational elements var(--color-primary)
--button-color Color used for buttons var(--link-color)
--space-unit The standard unit of space 1.6rem
--section-space The amount of space between sections calc(var(--space-unit) * 2)
--content-width The max width of the main content area 900px
--sidebar-width The max width of the sidebar 300px
